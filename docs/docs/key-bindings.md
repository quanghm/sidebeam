## Navigation

| Key | Action |
|---|---|
| `→` `↓` `Space` `l` `PgDn` | Next slide |
| `←` `↑` `k` `PgUp` | Previous slide |
| `Home` | First slide |
| `End` | Last slide |
| `g` + number + `Enter` | Go to slide |

## Presentation

| Key | Action |
|---|---|
| `s` | Cycle split mode (none / right / left) |
| `b` | Blank/unblank projector |
| `f` | Toggle projector fullscreen |

## Timer

| Key | Action |
|---|---|
| `p` | Pause/resume timer |
| `r` | Reset timer |

## App

| Key | Action |
|---|---|
| `h` | Toggle help |
| `⌘ + W` | Close presentation |
| `Esc` | Exit fullscreen / close help / cancel go-to |
| `q` | Quit |

## Welcome Screen

| Key | Action |
|---|---|
| `1` – `9`, `0` | Open recent file (1st through 10th) |
| `⌘ + O` | Open file picker |
| `⌘ + W` | Quit |
